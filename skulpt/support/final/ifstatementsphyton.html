How to print if statements in Phyton


message = "Today's activity" -- (the message will be starting with “Today’s activity” at the end when its executed )

print ("Please enter the temperature :\n") -- (it will print “Please enter the temperature” )

entered_value = input () -- (User will be prompted to enter a value) (

if int(entered_value) > 30: -- (if the entered value greater than 30)

message = message + "swimming" -- (today’s activity equals to swimming)

elif int(entered_value) > 20: -- (if the value is greater than 20 but less than 30)

message = message + "tennis" -- (todays’s activity equals to tennis)

else : message = message + "sleeping" -- (otherwise anything else equals to sleeping)

print message – (the message will be printed with appropriate sections)

Full Code


message = "Today's activity";
print ("Please enter the temperature :\n");
entered_value = input ();
if int(entered_value) > 30:
  message = message + "swimming";
elif int(entered_value) > 20:
  message = message + "tennis";
else : message = message + "sleeping";
print message;
