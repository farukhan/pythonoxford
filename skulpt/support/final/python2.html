In this section, we will learn how to use quick sort function. Quick sort function helps user to sort the lists in given order by the user. For instance a group of number that are listed randomly can be grouped from smaller to larger or larger to smaller using this function. We will show this as an example in this tutorial. The function first selects a number from the list this can be from any side but it is ideal to choose the first number in the list. This number is needed to split the list to put them in order. We will name this as split point in our program. In this case 54 will be our first number. The program will look for a number that is larger than 54 and then 54 and then it will stop. It will then look at the right section of the list and stop once it finds smaller number until the numbers are listed from smaller to largest.

Example CODE - there are three different functions in the program those are quick sort, quicksort helper and partition. 1. Quick sort function defines the list of number that needs to be put in order. 2. Quick sort helper function is active when length of the list is less than or equal to one it’s already sorted. However if it is greater than it can be portioned to look for a smaller number. 3. The partition function splits the numbers in accordance with the rules given in if statements.
